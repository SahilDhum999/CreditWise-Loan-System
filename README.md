# CreditWise-Loan-System



# Problem Statement

Traditional loan approval systems in banks are mostly manual, time-consuming, and sometimes biased. Financial institutions face challenges in identifying genuine customers while reducing the risk of approving high-risk applicants. This project aims to build an intelligent machine learning-based loan approval prediction system that analyzes customer financial and personal details to predict whether a loan should be approved or rejected accurately and efficiently. 

# Key Points / Objectives

1. Built an end-to-end supervised Machine Learning pipeline for loan approval prediction using KNN, Logistic Regression, and Naive Bayes.
2. Performed Exploratory Data Analysis (EDA), data preprocessing, and feature engineering to improve model performance.
3. Implemented binary classification to predict whether a loan application should be approved or rejected.
4. Evaluated models using performance metrics such as Precision, Recall, Accuracy, and F1-Score.
5. Analyzed applicant details like income, credit score, loan amount, employment status, and savings to support data-driven loan decisions.
6. Designed the system to provide faster, more accurate, and unbiased loan approval predictions for financial institutions.
